U ovoj vježbi potrebno je napraviti slijedeće točke:

1. Instaliraj git
https://git-scm.com/downloads

Provjera verzije
# git version

2. Postavi globalno git username
# git  config --global user.name "name"

3. Postavi globalno git email
# git  config --global user.email <email@domena.com>

4. Provjera parameatra
# git config --global --list
